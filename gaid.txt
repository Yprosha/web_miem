1) Скачать и установить node.js
    https://nodejs.org/en/download 
    После ребутнуть комп желательно
    Проверьте в терминале версию ноды и нпм (node -v   ///   npm -v)
    Если выводит, то всё ок

2) В терминале в корне проекта прописать npm install

3) После установки там же прописать запуск скрипта для старта приложения
    npm run dev (каждое изменение будет перезапускать сервер)
    npm run start (просто запустит сервер)

4) Дальше в браузере в поисковой строке прописать одно из (порт в env можете поменять)
    localhost:8080
    127.0.0.1:8080
    (или можете по ip компу)

5) Если не работает и вы сделали всё, как тут, то пишите мне (если не ребутали комп, то ребутните)
